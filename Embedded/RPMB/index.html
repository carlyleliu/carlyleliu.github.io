|йҳ…иҜ»йҮҸ:|иҜ„и®әж•°:

дҝЎжҒҜе®үе…Ё

дҝЎжҒҜе®үе…Ёзҡ„дёүдёӘеҹәжң¬зӣ®ж ҮжҳҜжңәеҜҶжҖ§пјҢе®Ңж•ҙжҖ§е’ҢеҸҜз”ЁжҖ§гҖӮ

  • жңәеҜҶжҖ§ж„Ҹе‘ізқҖеҸӘжңүжҺҲжқғе®һдҪ“жүҚиғҪйҳ…иҜ»е’ҢзҗҶи§ЈдҝқеҜҶзҡ„дҝЎжҒҜгҖӮжІЎжңүи®ҝй—®жқғйҷҗзҡ„е…¶д»–дәәж— жі•йҳ…иҜ»жҲ–зҗҶи§ЈжңәеҜҶдҝЎжҒҜпјӣ
  • е®Ңж•ҙжҖ§ж„Ҹе‘ізқҖиғҪеӨҹзЎ®дҝқдҝЎжҒҜеҸ—еҲ°дҝқжҠӨпјҢд»ҘйҳІжӯўжңӘз»ҸжҺҲжқғзҡ„жӣҙж”№пјҢдҝ®ж”№жҲ–еҲ йҷӨгҖӮдҝЎжҒҜзҡ„е®Ңж•ҙжҖ§еҢ…жӢ¬дҪҝз”ЁиҜҶеҲ«е’Ңи®ӨиҜҒзӯүж–№жі•зҡ„иө·жәҗпјҢе®Ңж•ҙжҖ§е’ҢжӯЈзЎ®жҖ§пјӣ
  • еҸҜз”ЁжҖ§ж„Ҹе‘ізқҖдҝЎжҒҜе§Ӣз»ҲеҸҜдҫӣжҺҲжқғз”ЁжҲ·дҪҝз”ЁгҖӮ

RPMBз®Җд»Ӣ

RPMBжҳҜReplay Protected Memory Block(йҮҚж”ҫдҝқжҠӨеҶ…еӯҳеқ—)зҡ„з®Җз§°пјҢжҳҜeMMCдёӯзҡ„дёҖдёӘе…·жңүе®үе…Ёзү№жҖ§зҡ„еҲҶеҢәгҖӮжӯӨеҠҹиғҪдҪҝи®ҫеӨҮиғҪеӨҹе°Ҷж•°жҚ®еӯҳеӮЁеңЁз»ҸиҝҮиә«д»ҪйӘҢиҜҒе№¶йҳІжӯўйҮҚж”ҫж”»еҮ»зҡ„е°ҸеһӢзү№е®ҡеҢәеҹҹ(йҖҡеёёжҳҜ4M Bytes)дёӯгҖӮиҝҷйҮҢж¶үеҸҠдёҖдёӘжҰӮеҝөReplay Attackе’ҢReplay ProtectedгҖӮ

Replay Attack(йҮҚж”ҫж”»еҮ»)

Aеҗ‘BиҜ·жұӮжңҚеҠЎ(жҜ”еҰӮиҜҙзҷ»еҪ•жҹҗдёӘзҪ‘з«ҷ)пјҢAе°ҶеҜҶз ҒhashеҢ–дј з»ҷBгҖӮдҪҶжҳҜеңЁиҝҷдёӯй—ҙпјҢEжҠ“еҸ–еҲ°иҜҘhashеҖјгҖӮжӯӨеҗҺпјҢEеҶ’е……Aеҗ‘BеҸ‘йҖҒеҗҢж ·зҡ„hashеҖјжқҘиҺ·еҸ–жңҚеҠЎгҖӮ

Replay Protected(йҮҚж”ҫдҝқжҠӨ)

  • еҠ йҡҸжңәж•°гҖӮиҜҘж–№жі•дјҳзӮ№жҳҜи®ӨиҜҒеҸҢж–№дёҚйңҖиҰҒж—¶й—ҙеҗҢжӯҘпјҢеҸҢж–№и®°дҪҸдҪҝз”ЁиҝҮзҡ„йҡҸжңәж•°пјҢеҰӮеҸ‘зҺ°жҠҘж–Үдёӯжңүд»ҘеүҚдҪҝз”ЁиҝҮзҡ„йҡҸжңәж•°пјҢе°ұи®ӨдёәжҳҜйҮҚж”ҫж”»еҮ»гҖӮзјәзӮ№жҳҜйңҖиҰҒйўқеӨ–дҝқеӯҳдҪҝз”ЁиҝҮзҡ„йҡҸжңәж•°пјҢиӢҘи®°еҪ•зҡ„ж—¶й—ҙж®өиҫғй•ҝпјҢеҲҷдҝқеӯҳе’ҢжҹҘиҜўзҡ„ејҖй”ҖиҫғеӨ§гҖӮ
  • еҠ ж—¶й—ҙжҲігҖӮиҜҘж–№жі•дјҳзӮ№жҳҜдёҚз”ЁйўқеӨ–дҝқеӯҳе…¶д»–дҝЎжҒҜгҖӮзјәзӮ№жҳҜи®ӨиҜҒеҸҢж–№йңҖиҰҒеҮҶзЎ®зҡ„ж—¶й—ҙеҗҢжӯҘпјҢеҗҢжӯҘи¶ҠеҘҪпјҢеҸ—ж”»еҮ»зҡ„еҸҜиғҪжҖ§е°ұи¶Ҡе°ҸгҖӮдҪҶеҪ“зі»з»ҹеҫҲеәһеӨ§пјҢи·Ёи¶Ҡзҡ„еҢәеҹҹиҫғе№ҝж—¶пјҢиҰҒеҒҡеҲ°зІҫзЎ®зҡ„ж—¶й—ҙеҗҢжӯҘе№¶дёҚжҳҜеҫҲе®№жҳ“гҖӮ
  • еҠ жөҒж°ҙеҸ·гҖӮе°ұжҳҜеҸҢж–№еңЁжҠҘж–Үдёӯж·»еҠ дёҖдёӘйҖҗжӯҘйҖ’еўһзҡ„ж•ҙж•°пјҢеҸӘиҰҒжҺҘж”¶еҲ°дёҖдёӘдёҚиҝһз»ӯзҡ„жөҒж°ҙеҸ·жҠҘж–Ү(еӨӘеӨ§жҲ–еӨӘе°Ҹ)пјҢе°ұи®Өе®ҡжңүйҮҚж”ҫеЁҒиғҒгҖӮиҜҘж–№жі•дјҳзӮ№жҳҜдёҚйңҖиҰҒж—¶й—ҙеҗҢжӯҘпјҢдҝқеӯҳзҡ„дҝЎжҒҜйҮҸжҜ”йҡҸжңәж•°ж–№ејҸе°ҸгҖӮзјәзӮ№жҳҜдёҖж—Ұж”»еҮ»иҖ…еҜ№жҠҘж–Үи§ЈеҜҶжҲҗеҠҹпјҢе°ұеҸҜд»ҘиҺ·еҫ—жөҒж°ҙеҸ·пјҢд»ҺиҖҢжҜҸж¬Ўе°ҶжөҒж°ҙеҸ·йҖ’еўһж¬әйӘ—и®ӨиҜҒз«ҜгҖӮ
  • дёҖж¬ЎжҖ§еҸЈд»ӨгҖӮ

RPMBи®ӨиҜҒ

RPMBдҪҝз”ЁеҜ№з§°з§ҳй’Ҙиә«д»Ҫи®ӨиҜҒпјҢд№ҹе°ұжҳҜhostе’ҢdeviceдҪҝз”ЁзӣёеҗҢзҡ„иә«д»ҪйӘҢиҜҒз§ҳй’ҘпјҢжӯӨз§ҳй’Ҙз§°д№Ӣдёәи®ӨиҜҒз§ҳй’Ҙ(AuthKey)д№ҹеҸҜз§°дёәRPMB KeyпјҢе·ҘдҪңж–№ејҸеҰӮдёӢпјҡ

  • AuthKeyйҖҡиҝҮhostеҶҷе…ҘеҲ°eMMCдёӯ(еңЁе®үе…Ёзҡ„зҺҜеўғдёӯеҶҷе…Ҙ)зҡ„OTPеҢәеҹҹдёӯ
  • hostе’ҢdeviceеңЁиҜ»еҶҷRPMBеҢәеҹҹзҡ„ж—¶еҖҷйңҖиҰҒзӯҫеҗҚе’ҢйӘҢиҜҒ
  • еҜ№ж¶ҲжҒҜиҝӣиЎҢзӯҫеҗҚж¶үеҸҠж¶ҲжҒҜиә«д»ҪйӘҢиҜҒд»Јз Ғ(Message Authentication Code)MACпјҢMACжҳҜhashеҖјпјҢMACжҳҜз”ұйҖҡиҝҮAuthKeyеҜ№Message + Write CounterдҪҝз”ЁHMAC SHA-256з®—жі•зӯҫеҗҚеҗҺеҫ—еҲ°
  • еҸ‘йҖҒзҡ„ж¶ҲжҒҜдёәMAC + Message + Write Counter

HMACжҳҜеҜҶй’Ҙзӣёе…ізҡ„е“ҲеёҢиҝҗз®—ж¶ҲжҒҜи®ӨиҜҒз ҒпјҢHMACиҝҗз®—еҲ©з”Ёе“ҲеёҢз®—жі•пјҢд»ҘдёҖдёӘеҜҶй’Ҙе’ҢдёҖдёӘж¶ҲжҒҜдёәиҫ“е…ҘпјҢз”ҹжҲҗдёҖдёӘж¶ҲжҒҜж‘ҳиҰҒдҪңдёәиҫ“еҮәгҖӮ дҝқжҠӨRPMBж•°жҚ®зҡ„е…ій”®е°ұжҳҜдҝқжҠӨAuthKeyпјҢеӣ жӯӨеңЁиҜ»еҶҷRPMBзҡ„ж—¶еҖҷпјҢйңҖиҰҒеңЁTEEзҡ„зҺҜеўғдёӯпјҢеӣ дёәеңЁи®ҝй—®RPMBпјҢйңҖиҰҒжҳҫејҸзҡ„дҪҝз”ЁAuthKeyпјҲеҰӮжһңдёҚеңЁTEEзҺҜеўғдёӯпјҢеҸҜд»ҘеҜ№AuthKeyиҝӣиЎҢеҠ еҜҶпјҢз„¶еҗҺдҪҝз”Ёи§ЈеҜҶжЁЎеқ—еҺ»ж“ҚдҪңпјүгҖӮ еҸҰеӨ–пјҢAuthKeyжңҖеҘҪд№ҹжҳҜзӢ¬дёҖж— дәҢзҡ„пјҢд№ҹе°ұжҳҜдёҺSoc UIDзӣёе…ізҡ„пјҢеҪўжҲҗиҮӘе·ұзӢ¬дёҖж— дәҢзҡ„KeyгҖӮ

RPMBжҳҜжҖҺд№Ҳе®һзҺ°йҮҚж”ҫдҝқжҠӨ

йҮҚж”ҫдҝқжҠӨжңҖеҹәжң¬зҡ„е°ұжҳҜиҰҒдҝқиҜҒжҜҸдёӘж¶ҲжҒҜжҳҜзӢ¬дёҖж— дәҢзҡ„пјҢRPMBеҲҶеҲ«еҠ е…Ҙ2дёӘеӣ еӯҗиҝӣеҺ»пјҡ

  • еңЁеҶҷж“ҚдҪңзҡ„ж—¶еҖҷпјҢдҪҝз”ЁWrite Counter(32Bytes)жқҘе®һзҺ°йҮҚж”ҫдҝқжҠӨгҖӮWrite CounterжҳҜз”ұDeviceз®ЎзҗҶзҡ„пјҢиҜҘи®Ўж•°еҷЁеңЁжҜҸж¬Ўжңүж•Ҳзҡ„йӘҢиҜҒеҶҷе…Ҙж¶ҲжҒҜеҗҺйҖ’еўһпјҢе…¶ж–°еҖје°ҶеҢ…еҗ«еңЁиҰҒеҸ‘йҖҒзҡ„дёӢдёҖдёӘиә«д»ҪйӘҢиҜҒд»Јз ҒMACзҡ„и®Ўз®—дёӯпјӣ
  • еңЁиҜ»ж“ҚдҪңзҡ„ж—¶еҖҷпјҢдҪҝз”ЁйҡҸжңәж•°Nonce(16Bytes)жқҘе®һзҺ°йҮҚж”ҫдҝқжҠӨгҖӮNonceжҳҜз”ұHostдә§з”ҹгҖӮ

дёҖдёӘи®ҝй—®RPMBе‘Ҫд»ӨйҖҡиҝҮHMAC SHA-256и®Ўз®—иҝӣиЎҢйӘҢиҜҒпјҢд»ҘдёӢеҮ йЎ№еҒҡдёәиҜҘи®Ўз®—зҡ„иҫ“е…Ҙпјҡ

  • иә«д»ҪйӘҢиҜҒзҡ„з§ҳй’ҘAuthKeyпјӣ
  • ж¶ҲжҒҜпјҢеҢ…жӢ¬е‘Ҫд»ӨжҲ–иҝ”еӣһзҡ„еҖјпјӣ
  • еҶҷи®Ўж•°еҷЁпјҢд№ҹе°ұжҳҜжҖ»зҡ„еҶҷRPMBзҡ„ж¬Ўж•°пјҢеӯҳж”ҫи®Ўж•°еҷЁзҡ„еҖјдёә32bitпјҢеҪ“еҶҷе…Ҙж¬Ўж•°иҫҫеҲ°0xFFFFFFFFзҡ„ж—¶еҖҷпјҢRPMBе°ұеҸҳдёәеҸӘиҜ»пјӣ
  • дёҖдёӘйҡҸжңәж•°пјҢе®ғжҳҜдёәжҜҸдёӘreadе‘Ҫд»ӨйҡҸжңәз”ҹжҲҗзҡ„ж•°еӯ—(Readзҡ„ж—¶еҖҷжүҚз”ЁеҲ°)гҖӮ е…·дҪ“зҡ„RPMBе‘Ҫд»Өж јејҸеҰӮдёӢпјҡ

д»ҺRPMBиҜ»ж•°жҚ®

  • еңЁHostз«ҜпјҢеҸ‘йҖҒиҜ»иҜ·жұӮд»ҘеҸҠдёҖдёӘ16BytesйҡҸжңәж•°Nonceз»ҷDeviceз«Ҝ
  • еңЁDeviceз«Ҝ(eMMC)пјҢе°ҶиҜ·жұӮзҡ„ж•°жҚ®д»ҺRPMBеҸ–еҮәжқҘпјҢйҖҡиҝҮAuthKeyеҜ№Data + йҡҸжңәж•°дҪҝз”ЁHMAC SHA-256з®—жі•зӯҫеҗҚеҗҺеҫ—еҲ°MACпјҢдёҺData + йҡҸжңәж•°дёҖиө·еҸ‘йҖҒз»ҷHostз«Ҝ
  • еңЁHostз«Ҝж”¶еҲ°DevcieеҸ‘йҖҒиҝҮжқҘзҡ„ж•°жҚ®пјҢ йҰ–е…ҲжҜ”иҫғйҡҸжңәж•°жҳҜеҗҰдёҺиҮӘе·ұдёҖиҮҙпјҢеҰӮжһңдёҖиҮҙпјҢеңЁHostз«ҜдҪҝз”ЁиҮӘе·ұзҡ„AuthKeyеҜ№жҺҘж”¶еҲ°зҡ„Data + йҡҸжңәж•°дҪҝз”ЁHMAC SHA-256з®—жі•зӯҫеҗҚеҗҺеҫ—еҲ°MAC1пјҢжҜ”иҫғиҮӘе·ұз”ҹжҲҗзҡ„MAC1дёҺжҺҘж”¶еҲ°зҡ„MACжҳҜеҗҰдёҖиҮҙгҖӮеҰӮжһңдёҖиҮҙпјҢHostз«Ҝе°ұеҸҜд»ҘзЎ®е®ҡиҜҘDataжҳҜд»ҺRPMBиҜ»еҸ–еҲ°зҡ„пјҢжІЎжңүиў«дҝ®ж”№иҝҮпјҢиҖҢдёҚжҳҜж”»еҮ»иҖ…дјӘйҖ зҡ„ж•°жҚ®гҖӮ

еҶҷж•°жҚ®еҲ°RPMB

еңЁ Host Socз«Ҝпјҡ

  • Hostз«ҜдҪҝз”ЁAuthKeyеҜ№Message(Data + Metadata) + Write Counter дҪҝз”ЁHMAC SHA-256з®—жі•зӯҫеҗҚеҗҺеҫ—еҲ°MAC
  • Hostз«Ҝе°ҶMAC + Message + Write Counterдј з»ҷDevice

еңЁDevicesз«Ҝ(eMMC/UFS)пјҡ

  • DeviceжҺҘеҸ—еҲ°Hostдј йҖҒиҝҮжқҘзҡ„MAC + Message + Write Counter
  • Deviceз«ҜжҜ”иҫғжҺҘж”¶еҲ°зҡ„Write CounterжҳҜеҗҰдёҺиҮӘе·ұзҡ„дёҖиҮҙ
  • Devicesе°ҶжҺҘеҸ—еҲ°зҡ„Message + Write Counter + devciesз«Ҝзҡ„AuthKey з»ҸиҝҮSHA-256з”ҹжҲҗMAC1
  • е°ҶDevicesз«Ҝз”ҹжҲҗзҡ„MAC1дёҺжҺҘж”¶еҲ°зҡ„MACиҝӣиЎҢжҜ”иҫғпјҢеҰӮжһңдёӨиҖ…дёҖиҮҙпјҢе°ұе°ҶDataеҶҷе…ҘеҲ°RPMBдёӯ

йҖҡиҝҮдёҠиҝ°зҡ„еҶҷе…ҘжөҒзЁӢпјҢеҸҜд»ҘдҝқиҜҒ RPMB дёҚдјҡиў«йқһжі•зҜЎж”№гҖӮ

дҪҝз”Ёmmcе‘Ҫд»Өи®ҝй—®RPMB

дҪҝз”Ёmmcе‘Ҫд»ӨеҸҜд»Ҙи®ҝй—®RPMBпјҢжҹҘзңӢи®ҝй—®RPMBзӣёе…ізҡ„з”Ёжі•еҰӮдёӢпјҡ

  • еҲӣе»әAuthKeyе’ҢWrongAuthKey

  • е°ҶAuthKeyеҶҷе…ҘеҲ°eMMCзҡ„OTPдёӯ

  • еҲӣе»әдёҖдёӘ256Bytesзҡ„ж•°жҚ® д№ӢжүҖд»ҘиҰҒеҲӣе»ә256Bytesзҡ„ж•°жҚ®пјҢжҳҜеӣ дёәmmcи§„е®ҡиҰҒеҶҷ256Bytesпјҡmmc rpmb write-block

    <256 byte data file>

  • з”ЁжӯЈзЎ®зҡ„AuthKeyеҫҖRPMBеҶҷе…Ҙж•°жҚ®

    еңЁиҝҷдёӘжӯҘйӘӨдёӯпјҢжІЎжңүзңӢеҲ°жңүиҜ»еҸ–Write Counterзҡ„еҠЁдҪңпјҢиҝҷжҳҜеӣ дёәmmcе‘Ҫд»Өе·Із»ҸеңЁwrite-blockдёӯйҰ–е…Ҳе®һзҺ°дәҶиҜ»еҸ–Write CounterпјҢиҜҰи§Ғmmcжәҗз Ғdo_rpmb_write_block()еҮҪж•°пјҢиҜҘеҮҪж•°дјҡи°ғз”Ёrpmb_read_counter()иҺ·еҸ–Write CounterгҖӮ

  • з”Ёй”ҷиҜҜзҡ„AuthKeyеҫҖRPMBеҶҷе…Ҙж•°жҚ®

    иҝҷж—¶еҖҷдјҡжҸҗзӨәRPMB operation failed, retcode 0x0002зҡ„й”ҷиҜҜпјҢиЎЁзӨәйӘҢиҜҒеӨұиҙҘгҖӮ
