Linux驱动之PWM

发表于 2020-09-02 更新于 2024-06-03 分类于 Linux ， 驱动

PWM概述

  • Ton：信号高电平持续时间
  • Toff：信号底电平持续时间
  • Period： 完整pwm周期
  • Duty cycle：pwm信号周期内保持为ON的时间百分比

Linux下PWM驱动

数据结构

pwm的数据结构如下图所示：原图

  • pwm_chip结构体用于抽象soc的pwm控制器
  • pwm_device用来抽象设备
  • pwm_state表示pwm设备的周期、占空比、极性等信息
  • pwm ops结构体提供一系列回调函数。这些回调函数的操作对象是具体的pwm device
