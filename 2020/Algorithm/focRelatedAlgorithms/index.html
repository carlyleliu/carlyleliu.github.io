MTPA

дёәд»Җд№ҲиҰҒз”Ё MTPA

еҪ“з”өжңәйҮҮз”Ё id=0 зҡ„жҺ§еҲ¶зӯ–з•ҘпјҢиҝҷз§ҚжҺ§еҲ¶ж–№жі•еҝҪз•ҘдәҶзЈҒйҳ»иҪ¬зҹ©зҡ„дҪңз”ЁпјҢз”өзЈҒиҪ¬й’ңж–№зЁӢеҰӮдёӢпјҡ

\[ \tau _e = \frac{3}{2}p[k_e \cdot i_q + (L_d - L_q)\cdot i_d \cdot i_q] \]

иҪ¬зҹ©еҲҶдёәж°ёзЈҒиҪ¬зҹ© Tr е’ҢзЈҒйҳ»иҪ¬зҹ© TmпјҢиҖҢ id=0 еҸӘеү©дёӢ TrгҖӮиҝҷдјҡеҜјиҮҙз”өжөҒзҡ„еҲ©з”ЁзҺҮдёҚй«ҳпјҢзі»з»ҹзҡ„ж•ҲзҺҮйҷҚдҪҺгҖӮжүҖд»Ҙ id=0 зҡ„жҺ§еҲ¶жҜ”иҫғйҖӮз”ЁдәҺйҡҗжһҒејҸз”өжңәпјҲLd=LqпјүпјҢиҖҢеҜ№дәҺеҮёжһҒејҸз”өжңәе№¶дёҚжңҖдјҳпјҢжүҖд»ҘйңҖиҰҒйҮҚж–°иҖғиҷ‘жҺ§еҲ¶зӯ–з•ҘгҖӮ

жҺЁеҜјиҝҮзЁӢ

з”өеҠЁжңәз”өеҺӢж–№зЁӢпјҡ

\[ \begin{aligned} U_d = rI_d - L_qI_q\omega_e \\ U_q = rI_q + k_E\omega_e + L_dI_d\omega_e \end{aligned} \]

йӮЈд№Ҳз”өеҠЁжңәж¶ҲиҖ—еҜ№жңүеҠҹеҠҹзҺҮдёәпјҡ

\[ P = \frac{3}{2}(U_dI_d + U_qI_q) \]

е°Ҷз”өеҠЁжңәж–№зЁӢд»Је…ҘеҠҹзҺҮж–№зЁӢеҫ—пјҡ

\[ P = \frac{3}{2} r (I_d^2 + I_q^2) + [k_EI_q + (L_d - L_q)I_dI_q]\omega_e \]

з”өеҠЁжңәзҡ„жңүеҠҹеҠҹзҺҮдёҖйғЁеҲҶж¶ҲиҖ—еңЁз»•йҳ»з”өйҳ»дёҠеҸ‘зғӯпјҢеҸҰдёҖйғЁеҲҶз”ЁдәҺиҫ“еҮәжңәжў°еҠҹзҺҮпјҡ

з”өеҠЁжңәзҡ„жңәжў°еҠҹзҺҮдёәпјҡ

\[ P_{mech} = \tau_e \omega_{mech} \]

жңәжў°иҪ¬йҖҹдёҺз”өйў‘зҺҮд№Ӣй—ҙзҡ„е…ізі»дёәпјҡ

\[ \omega_e = p \omega_{mech} \]

жүҖд»Ҙпјҡ

\[ \tau_e \omega_{mech} = \frac{3}{2} r (I_d^2 + I_q^2) + [k_EI_q + (L_d - L_q)I_dI_q] p \omega_{mech} \]

еҫ—еҲ°з”өзЈҒиҪ¬й’ңе…¬ејҸпјҡ

\[ \tau _e = \frac{3}{2}p[k_e \cdot i_q + (L_d - L_q)\cdot i_d \cdot i_q] \]

еҰӮжһңз»•йҳ»дёӯз”өжөҒеі°еҖјжҳҜ I_s

\[ I_s^2 = I_d^2 + I_q^2 \]

еҪ“иҫ“еҮәз”өзЈҒиҪ¬й’ңдёҖе®ҡж—¶пјҡ

\[ I_q = \frac{\tau_e}{\frac{3}{2}p[(k_E) + (L_d - L_q)I_d]} \]

иҒ”з«ӢдёҠдёӨејҸеҫ—пјҡ

\[ I_s^2 = I_d^2 + \left ( \frac{\tau_e}{\frac{3}{2}p[(k_E) + (L_d - L_q)I_d]} \right )^2 \]

дёҠејҸеҸҜд»ҘзңӢдҪңжҳҜдёҖдёӘе…ідәҺ\(I_d\)зҡ„еҮҪж•°пјҢдёҠејҸжңүжңҖе°ҸеҖјпјҢеҪ“з”өзЈҒиҪ¬й’ңдёҖе®ҡж—¶пјҢжңүдёҖдёӘжңҖе°Ҹеі°еҖјз”өжөҒ\(I_s\)пјҢеҸ–\(I_s\)жңҖе°Ҹж—¶жңүпјҡ

\[ \begin{aligned} \frac{\partial I_s^2}{\partial I_d} &= 2I_d + 2I_q \frac{\tau_e}{\frac{3}{2}p} \frac{-(L_d - L_q)}{[k_E + (L_d - L_q)I_d]^2} \\ & = 2I_d + 2I_q \frac{\tau_e}{\frac{3}{2}p[k_E + (L_d - L_q)I_d]} \frac{-(L_d - L_q)}{[k_E + (L_d - L_q)I_d]} \\ & = 2I_d + 2I_q^2\frac{-(L_d - L_q)}{[k_E + (L_d - L_q)I_d]} = 0 \end{aligned} \]

ж•ҙзҗҶеҫ—пјҡ

\[ (L_d - L_q)I_d^2 + k_EI_d - (L_d - L_q)I_q^2 = 0 \]

жұӮи§Ј\(I_d\)еҫ—еҲ°пјҡ

\[ I_d = - \frac{k_E}{2(L_d - L_q)} \pm \sqrt{\left ( \frac{k_E}{2(L_d - L_q)} \right )^2 + I_q^2} \]

еҰӮжһң\(L_d < L_q\)ж—¶еҪ“з”өзЈҒиҪ¬й’ңдёҖе®ҡж—¶з”өжөҒеі°еҖј\(I_s\)жңҖе°Ҹпјҡ

\[ I_d = - \frac{k_E}{2(L_d - L_q)} - \sqrt{\left ( \frac{k_E}{2(L_d - L_q)} \right )^2 + I_q^2} \]

ејұзЈҒжҺ§еҲ¶

з®Җд»Ӣ

ејұзЈҒжҺ§еҲ¶зҡ„иҝҷдёӘжҖқжғіжәҗиҝҳжҳҜжқҘиҮӘд»–еҠұзӣҙжөҒз”өеҠЁжңәзҡ„и°ғзЈҒжҺ§еҲ¶гҖӮеҪ“д»–еҠұзӣҙжөҒз”өеҠЁжңәз«Ҝз”өеҺӢиҫҫеҲ°жңҖеӨ§з”өеҺӢж—¶пјҢеҸӘиғҪйҖҡиҝҮи°ғиҠӮз”өжңәзҡ„еҠұзЈҒз”өжөҒпјҢиҝӣиҖҢж”№еҸҳеҠұзЈҒзЈҒйҖҡпјҢеңЁдҝқиҜҒиҫ“еҮәз”өеҺӢжңҖеӨ§еҖјдёҚеҸҳзҡ„жқЎд»¶дёӢпјҢдҪҝз”өжңәиғҪжҒ’еҠҹзҺҮиҝҗиЎҢдәҺжӣҙй«ҳзҡ„иҪ¬йҖҹгҖӮд№ҹе°ұжҳҜиҜҙпјҢд»–еҠұзӣҙжөҒз”өеҠЁжңәеҸҜд»ҘйҖҡиҝҮйҷҚдҪҺеҠұзЈҒз”өжөҒиҫҫеҲ°ејұзЈҒжү©йҖҹзҡ„зӣ®зҡ„гҖӮеҜ№дәҺ PMSM иҖҢиЁҖпјҢеҠұзЈҒзЈҒеҠЁеҠҝеӣ ж°ёзЈҒдҪ“дә§з”ҹиҖҢж— жі•и°ғиҠӮпјҢеҸӘиғҪйҖҡиҝҮи°ғиҠӮе®ҡеӯҗз”өжөҒпјҢеҚіеўһеҠ е®ҡеӯҗзӣҙиҪҙеҺ»зЈҒз”өжөҒеҲҶйҮҸжқҘз»ҙжҢҒй«ҳйҖҹиҝҗиЎҢж—¶з”өеҺӢзҡ„е№іиЎЎгҖӮиҫҫеҲ°ејұзЈҒжү©йҖҹзҡ„зӣ®зҡ„гҖӮ

жҺЁеҜј

еҫҲеӨҡеә”з”ЁйңҖиҰҒй©¬иҫҫе·ҘдҪңеңЁй«ҳдәҺйўқе®ҡйҖҹеәҰзҡ„иҢғеӣҙеҶ…пјҢиҝҷйҮҢе°ұйңҖиҰҒејұзЈҒжҺ§еҲ¶жқҘе®һзҺ°пјҢз”өжөҒзҡ„з”өеҺӢзәҰжқҹжқЎд»¶еҰӮдёӢпјҡ

\[ (L_qI_q)^2 + (k_E + L_dI_d)^2 \leqslant \frac{U_{1-limit}^2}{\omega_e^2} \]

з”өжөҒе№…еҖјзәҰжқҹжқЎд»¶пјҡ

\[ I_d^2 + I_q^2 \leqslant I_{s-limit}^2 \]

MTPA дёӢжңү\(I_d\)пјҡ

\[ MTPA: I_d = \left\{\begin{matrix} - \frac{k_E}{2(L_d - L_q)} + sign(L_d - L_q) \sqrt{\left ( \frac{k_E}{2(L_d - L_q)} \right )^2 + I_q^2}, L_d \neq L_q\\ 0, L_d = L_q \end{matrix}\right. \]

е…¶дёӯпјҡ

\[ sign(x) = \left\{\begin{matrix} 1, x > 0\\ 0, x = 0\\ -1, x < 0 \end{matrix}\right. \]

ејұзЈҒжғ…еҶөдёӢжңү\(I_d\):

\[ I_d = \left\{\begin{matrix} -\frac{k_E}{L_d}+ \frac{\sqrt{(\frac{U_{1-limit}}{\omega_e})^2 - (L_qI_q)^2}}{L_d}, \frac{U_{1-limit}}{\omega_e} \geq L_qI_q\\ n/a, \frac{U_{1-limit}}{\omega_e} < L_qI_q \end{matrix}\right. \]

з”өжөҒеүҚйҰҲ

еүҚйҰҲзҗҶи®ә

еүҚйҰҲжҺ§еҲ¶жҳҜжҢүз…§жү°еҠЁйҮҸиҝӣиЎҢиЎҘеҒҝйғҪејҖзҺҜжҺ§еҲ¶пјҢеҪ“зі»з»ҹеҮәзҺ°жү°еҠЁж—¶пјҢжҢүз…§жү°еҠЁйҮҸеҪ“еӨ§е°ҸзӣҙжҺҘдә§з”ҹзҹ«жӯЈдҪңз”ЁгҖӮ еүҚйҰҲжҺ§еҲ¶з»“жһ„еӣҫеҰӮдёӢпјҡ

еӣҫдёӯ\(G_n(s)\)жҳҜиў«жҺ§еҜ№иұЎжү°еҠЁйҖҡйҒ“зҡ„дј йҖ’еҮҪж•°пјҢ\(D_n(s)\)жҳҜеүҚйҰҲжҺ§еҲ¶еҷЁзҡ„дј йҖ’еҮҪж•°пјҢ\(G(s)\)дёәиў«жҺ§еҜ№иұЎжҺ§еҲ¶йҖҡйҒ“дј йҖ’еҮҪж•°пјҢnгҖҒuгҖҒy еҲҶеҲ«дёәжү°еҠЁйҮҸгҖҒжҺ§еҲ¶йҮҸе’Ңиҫ“еҮәйҮҸ еҒҮе®ҡиҫ“е…ҘйҮҸ u1=0пјҢеҲҷжңүпјҡ

\[ Y(s) = Y_1(s) + Y_2(s) = [D_n(s)G(s) + G_n(s)]N(s) \]

иӢҘдҪҝеүҚйҰҲжҺ§еҲ¶дҪңз”Ёе®Ңе…ЁиЎҘеҒҝжү°еҠЁдҪңз”ЁпјҢеҲҷеә”дҪҝ\(Y(s) = 0\)еҚіпјҡ

\[ D_n(s)G(s) + G_n(s) = 0 \]

жүҖд»ҘеүҚйҰҲжҺ§еҲ¶еҷЁзҡ„дј йҖ’еҮҪж•°дёәпјҡ

\[ D_n(s) = - \frac{G_n(s)}{G(s)} \]

еңЁе®һйҷ…еә”з”ЁдёӯгҖӮеӣ дёәеүҚйҰҲжҺ§еҲ¶дёәдёҖдёӘејҖзҺҜзі»з»ҹгҖӮеӣ жӯӨеёёеёёйҮҮз”ЁеҸҚйҰҲ+еүҚйҰҲзҡ„еӨҚеҗҲжҺ§еҲ¶ж–№ејҸгҖӮ

з”өжңәжҺ§еҲ¶дёӯзҡ„еүҚйҰҲжҺ§еҲ¶

еңЁз”өжңәжҺ§еҲ¶дёӯеҠ е…ҘеүҚйҰҲжҺ§еҲ¶пјҢе…¶з»“жһ„жЎҶеӣҫеҰӮдёӢпјҡ
