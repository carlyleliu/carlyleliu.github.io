FOCз”өжөҒйҮҮж ·

еҸ‘иЎЁдәҺ 2022-10-06 жӣҙж–°дәҺ 2024-06-03 еҲҶзұ»дәҺ йЎ№зӣ® пјҢ жңәеҷЁдәә йҳ…иҜ»ж¬Ўж•°пјҡ Disqusпјҡ жң¬ж–Үеӯ—ж•°пјҡ 1.5k йҳ…иҜ»ж—¶й•ҝ ≈ 5 еҲҶй’ҹ

йҮҮж ·зӮ№

еңЁеҹәе°”йңҚеӨ«е®ҡеҫӢдёӢпјҢдёүзӣёз”өжөҒзҡ„еҗҲеә”иҜҘзӯүдәҺ0пјҢеӣ жӯӨеҸӘйңҖиҰҒиҺ·еҸ–дә®зӣёз”өжөҒе°ұеҸҜд»ҘйҮҚжһ„еҮәе®Ңж•ҙзҡ„дёүзӣёз”өжөҒгҖӮ

еҜ№дәҺSTM32дёҖиҲ¬йҖҡиҝҮй«ҳзә§е®ҡж—¶еҷЁзҡ„channel4дҪңдёәADCзҡ„и§ҰеҸ‘жәҗпјҢеҜ№дәҺдёӢжЎҘиҮӮз”өжөҒйҮҮж ·пјҢйңҖиҰҒеңЁдёӢжЎҘиҮӮMOSз®ЎеҜјзӯ’зҡ„ж—¶еҖҷжүҚиғҪеҺ»йҮҮж ·пјҢиҖҢдё”йңҖиҰҒеңЁMOSз®ЎеҜјйҖҡж—¶й—ҙд»ҘеҸҠз”өжөҒзЁіе®ҡеҗҺжүҚиғҪйҮҮж ·еҲ°жҜ”иҫғеҸҜйқ зҡ„з”өжөҒгҖӮ

  • еҪ“ \(\Delta_1\) еӨ§дәҺдёӨеҖҚзҡ„ $T_{dead} + T_{on} + T_{ring} + T_{conv} $ йӮЈд№ҲеңЁPWM Freq Middle ж—¶еҲ»еҮәеҸ‘ADCиҪ¬жҚўд№ҹжҳҜи¶іеӨҹиҪ¬жҚў3зӣёдёӯд»»ж„Ҹзҡ„дёҖзӣё
  • еҪ“ \(\Delta_1\) еӨ§дәҺ$T_{dead} + T_{on} + T_{ring} + T_{conv} йӮЈд№Ҳд»Һ CCRmaxд№ӢеҗҺејҖе§ӢиҪ¬жҚўд№ҹжҳҜеҸҜд»ҘиҪ¬жҚў3зӣёдёӯд»»ж„Ҹзҡ„дёҖзӣё
  • еҪ“ \(\Delta_2\) > \(\Delta_0\) > \(\Delta_1\) йӮЈд№ҲеҜјйҖҡжңҖзҹӯзҡ„йӮЈдёҖзӣёе°ұжІЎжңүи¶іеӨҹзҡ„иҪ¬жҚўж—¶й—ҙдәҶпјҢйӮЈд№ҲеҸӘиғҪиҪ¬жҚўеҜјйҖҡж—¶й—ҙй•ҝзҡ„йӮЈдёӨзӣё
  • \(\Delta_1\) < \(\Delta_0\) е’Ң \(\Delta_2\) < \(\Delta_0\) жІЎжңүи¶іеӨҹзҡ„иҪ¬жҚўж—¶й—ҙпјҢжңүдёӨзӣёз”өжөҒдёҚеҸҜиҺ·еҸ–е°Ҷж— жі•йҮҚе»әдёүзӣёз”өжөҒ

еӣ жӯӨйңҖиҰҒж №жҚ®uгҖҒvгҖҒwдёүзӣёpwmзҡ„еҚ з©әжҜ”жқҘи°ғж•ҙtimerзҡ„channel 4жҜ”иҫғеҖјжқҘи°ғж•ҙи§ҰеҸ‘ж—¶еҲ»е®ҢжҲҗдёүзӣёз”өжөҒзҡ„йҮҚе»әе·ҘдҪңгҖӮ

зӣёз”өжөҒйҮҚжһ„

  • еҜ№дәҺSector4е’Ң5пјҢCзӣёеҜјйҖҡж—¶й—ҙжңҖй•ҝпјҢзӣёеә”зҡ„е°ұжҳҜдёӢжЎҘиҮӮеҜјйҖҡж—¶й—ҙжңҖзҹӯпјҢиҝҷдёӘж—¶еҖҷеә”иҜҘйҮҮйӣҶAе’ҢBзӣёз”өжөҒ
  • еҜ№дәҺSector2е’Ң3пјҢBзӣёеҜјйҖҡж—¶й—ҙжңҖй•ҝпјҢзӣёеә”зҡ„е°ұжҳҜдёӢжЎҘиҮӮеҜјйҖҡж—¶й—ҙжңҖзҹӯпјҢиҝҷдёӘж—¶еҖҷеә”иҜҘйҮҮйӣҶAе’ҢCзӣёз”өжөҒ
  • еҜ№дәҺSector1е’Ң6пјҢAзӣёеҜјйҖҡж—¶й—ҙжңҖй•ҝпјҢзӣёеә”зҡ„е°ұжҳҜдёӢжЎҘиҮӮеҜјйҖҡж—¶й—ҙжңҖзҹӯпјҢиҝҷдёӘж—¶еҖҷеә”иҜҘйҮҮйӣҶBе’ҢCзӣёз”өжөҒ

еёёз”ЁиҠҜзүҮ

MP6540з”өжөҒйҮҮж ·

MP6540H зҡ„еҶ…йғЁз”өжөҒйҮҮж ·з”өи·ҜгҖӮе…¶иҫ“еҮәеҸҜйҖҡиҝҮеӨ–йғЁз”өйҳ»еҷЁ (RTERM) е’ҢеҸӮиҖғз”өеҺӢ (VREF) жқҘи®ҫзҪ®гҖӮдёӨдёӘзӯүеҖјз”өйҳ»иҝһжҺҘеҲ° ADC з”өжәҗе’Ңең°пјҢжҺҘең°з”ЁдәҺз»Ҳжӯўиҫ“еҮәгҖӮ

MP6540 зҡ„з”өжөҒйҮҮж ·и®Ўз®—жҜ”иҫғз®ҖеҚ•пјҢиҝҷйҮҢе°ұдёҚеҶҚиҜҰз»ҶиҜҙжҳҺдәҶгҖӮ

DRV8301/DRV8302 з”өжөҒйҮҮж ·

йҰ–е…Ҳе°ҶDRV8301зҡ„жЎҶеӣҫеҲ—еҮәжқҘпјҢеӨ§иҮҙдәҶи§ЈдёӢз»“жһ„пјҢдё»иҰҒе°ұжҳҜйҖҡиҝҮSNxе’ҢSPxеӣӣдёӘеј•и„ҡд№Ӣй—ҙзҡ„з”өйҳ»жқҘйҮҮж ·з”өжөҒгҖӮ

еҰӮжһңжҲ‘д»¬йңҖиҰҒйҮҮж ·дёҖдёӘжЎҘиҮӮзҡ„з”өжөҒпјҢйӮЈдј з»ҹзҡ„ж–№ејҸиҮӘз„¶жҳҜзӣҙжҺҘеңЁз”өи·ҜдёҠдёІиҒ”дёҖдёӘйҮҮж ·з”өйҳ»пјҢз„¶еҗҺйҮҮж ·дёӨз«Ҝз”өеҺӢеҚіеҸҜгҖӮз”ұдәҺз”өйҳ»жҳҜдёІиҒ”иҝӣз”өи·Ҝзҡ„пјҢжүҖд»Ҙйҳ»еҖјеҪ“з„¶жҳҜиҰҒи¶ҠдҪҺи¶ҠеҘҪгҖӮдҪҶжҳҜиҝҷеҸҲдјҡеёҰжқҘеҸҰдёҖдёӘй—®йўҳпјҢдҪҺйҳ»еҖјеҜјиҮҙзҡ„йҮҮж ·зІҫеәҰдёҚй«ҳпјҢеӣ жӯӨйңҖиҰҒиҝҗж”ҫжқҘе®һзҺ°з”өжөҒзҡ„йҮҮж ·гҖӮ

жҲ‘д»¬д»ҘodriverеҺҹзҗҶеӣҫдёәдҫӢиҝӣиЎҢи®Ўз®—пјҢеҰӮдёӢпјҡ

иҝҷйҮҢR27е’ҢR28з”өйҳ»е°ұжҳҜйҮҮж ·з”өйҳ»,\(500u\Omega(500 * 10^{-6})\Omega\).

дёӢйқўжҳҜе®ҳж–№жүӢеҶҢз»ҷеҮәзҡ„з”өжөҒи®Ўз®—ж–№ејҸгҖӮ

иҝҷйҮҢеҒҡеҰӮдёӢиҜҙжҳҺпјҡ \(SN_x - SN_x\) жҳҜйҮҮж ·з”өйҳ»дёӨз«Ҝзҡ„з”өеҺӢеҖјпјҢд№ҹжҳҜжҲ‘д»¬йңҖиҰҒжөӢйҮҸзҡ„еҖјпјҢеҫ—еҲ°иҜҘеҖјеҗҺйҷӨд»Ҙз”өйҳ»е°ұеҫ—еҲ°еҜ№еә”зҡ„з”өжөҒеҖјгҖӮVoжҳҜжҲ‘д»¬ADCеҫ—еҲ°зҡ„еҖјгҖӮй’ҲеҜ№е®ҳж–№з»ҷеҮәзҡ„е…¬ејҸпјҢжҲ‘д»¬еҒҡеҰӮдёӢи®Ўз®—пјҡ

\[ V_o = \frac{V_{REF}}{2} - G \times (SN_x - SP_x) \]

еҢ–з®ҖеҰӮдёӢпјҡ

\[ (SN_x - SP_x) = \frac{(\frac{V_{REF}}{2} - V_o )}{G} \]

иҝҷйҮҢGжҳҜ10гҖҒ20гҖҒ40гҖҒ80. \(V_{REF}\)дёҖиҲ¬жҳҜ3.3vпјҢйӮЈд№ҲеҰӮдёӢпјҡ

\[ (\frac{V_{REF}}{2} - V_o ) = (\frac{3.3}{2} - \frac{ADC}{2^{12}} \times 3.3 ) = \frac{2^{11} - ADC}{2^{12}} \times 3.3 \]

еҫ—еҲ°з”өжөҒеҖјеҰӮдёӢпјҡ

\[ (SN_x - SP_x) = I_{measure} \times R_{Isensor} = \frac{2^{11} - ADC}{2^{12}} \times \frac{3.3}{G} \]

еҫ—еҲ°жңҖз»Ҳз»“жһңдёәпјҡ

\[ I_{measure} = \frac{2^{11} - ADC}{2^{12}} \times \frac{3.3}{G} \times \frac{1}{R_{Isensor}} \]

жӯ»еҢәж—¶й—ҙзҡ„и®Ўз®—

жҲ‘д»¬з”ЁдёӢеҲ—е…¬ејҸи®Ўз®—жҺ§еҲ¶жӯ»еҢәж—¶й—ҙпјҡ

\[ t_{dead} = [(td_{offmax} - td_{onmin}) + (tpdd_{max} - tpdd_{min})] * 1.2 \]

  • \(td_{offmax}\) пјҡMOSз®ЎжңҖеӨ§е…іж–ӯе»¶иҝҹж—¶й—ҙ
  • \(td_{onmin}\)пјҡMOSз®ЎжңҖе°ҸејҖйҖҡе»¶иҝҹж—¶й—ҙ
  • \(tpdd_{max}\)пјҡй©ұеҠЁеҷЁжңҖеӨ§дј иҫ“е»¶иҝҹж—¶й—ҙ
  • \(tpdd_{min}\)пјҡй©ұеҠЁеҷЁжңҖе°Ҹдј иҫ“е»¶иҝҹж—¶й—ҙ

еңЁиҜҘе…¬ејҸдёӯпјҢз¬¬дёҖйЎ№\(td_{offmax}-td_{onmin}\)дёәжңҖеӨ§е…іж–ӯе»¶иҝҹж—¶й—ҙе’ҢжңҖе°ҸејҖйҖҡе»¶иҝҹж—¶й—ҙд№Ӣе·®гҖӮиҝҷдёҖйЎ№дё»иҰҒжҸҸиҝ°MOSз®ЎеҷЁд»¶з»“еҗҲжүҖз”Ёзҡ„й—ЁжһҒз”өйҳ»зҡ„зү№жҖ§гҖӮз”ұдәҺдёҠеҚҮе’ҢдёӢйҷҚж—¶й—ҙйҖҡеёёжҜ”е»¶иҝҹж—¶й—ҙзҹӯеҫҲеӨҡпјҢиҝҷйҮҢе°ұдёҚиҖғиҷ‘е®ғд»¬гҖӮеҸҰдёҖйЎ№\(tpdd_{max} - tpdd_{min}\)дёәз”ұй©ұеҠЁеҷЁеҶіе®ҡзҡ„дј иҫ“е»¶иҝҹж—¶й—ҙд№Ӣе·®пјҲе»¶иҝҹж—¶й—ҙдёҚеҢ№й…ҚпјүгҖӮиҜҘеҸӮж•°йҖҡеёёеҸҜеңЁй©ұеҠЁеҷЁеҲ¶йҖ е•ҶжҸҗдҫӣзҡ„й©ұеҠЁеҷЁж•°жҚ®иЎЁдёӯжҹҘжүҫеҲ°гҖӮеҜ№дәҺеҹәдәҺе…үиҖҰеҗҲеҷЁзҡ„й©ұеҠЁеҷЁпјҢиҜҘеҸӮж•°еҖјйҖҡеёёеҫҲеӨ§гҖӮ

еҜ№дәҺcsd18540q:

\(td_{offmax}\) = 20ns \(td_{onmin}\) = 6ns

STM32е®һзҺ°еҸҢADCйҮҮж ·дёүз”өйҳ»з”өжөҒ

йҮҚжһ„йҮҮж ·зӮ№

STM32зҡ„й«ҳзә§е®ҡж—¶еҷЁжңү6дёӘpwm channelпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”ЁеүҚдёүдёӘchannelиҫ“еҮәдә’иЎҘзҡ„pwmпјҢз¬¬еӣӣдёӘchannelз”ЁдәҺи§ҰеҸ‘adcзҡ„йҮҮж ·ж—¶еҲ»гҖӮиҝҷйҮҢдё»иҰҒжҳҜж №жҚ®жүҮеҢәд»ҘеҸҠдёүйҖҡйҒ“зҡ„pwmеҚ з©әжҜ”жқҘеҲӨж–ӯз”өжөҒйҮҮж ·зӮ№ж—¶еҲ»гҖӮ

йҮҚжһ„з”өжөҒйҮҮж ·

жҲ‘д»¬еҸҜд»Ҙж №жҚ®еҪ“еүҚжүҮеҢәд»ҘеҸҠдёүйҖҡйҒ“зҡ„pwmеҚ з©әжҜ”жқҘйҖүжӢ©йҮҮж ·е“ӘдёӨзӣёз”өжөҒпјҢйҖҡеёёдёҚйҮҮж ·еҚ з©әжҜ”жңҖе°Ҹзҡ„йӮЈдёҖйҖҡйҒ“пјҢиҖҢйҖҡиҝҮеҹәе°”йңҚеӨ«е®ҡеҫӢжқҘз®—з¬¬дёүйҖҡйҒ“зҡ„з”өжөҒпјҢйҒҝе…ҚйҮҮж ·еҢәй—ҙеӨӘе°ҸйҖ жҲҗзҡ„йҮҮж ·з”өжөҒдёҚзЁіе®ҡгҖӮе…·дҪ“з®—жі•еҰӮдёӢпјҡ

жіЁж„ҸиҝҷйҮҢsector 1зҡ„йҮҮж ·йҖҡйҒ“жҳҜADC1зҡ„MC_ADC_CHANNEL_2е’ҢADC2зҡ„MC_ADC_CHANNEL_12пјҢsector2зҡ„йҮҮж ·йҖҡйҒ“жҳҜADC1зҡ„MC_ADC_CHANNEL_1е’ҢADC2зҡ„MC_ADC_CHANNEL_12гҖӮиҝҷдёӘиЎЁж јжҳҜж №жҚ®еүҚйқўзҡ„зҗҶи®әйғЁеҲҶд»ҘеҸҠеҺҹзҗҶеӣҫжқҘи®Ўз®—еҫ—еҲ°зҡ„йҖҡйҒ“гҖӮ

еңЁеҫ—еҲ°еҺҹе§Ӣзҡ„ADCеҖјд№ӢеҗҺеҸҜд»ҘйҖҡиҝҮз”өи·ҜеҺҹзҗҶе°ҶADCиҪ¬жҚўдёәеҜ№еә”зҡ„з”өжөҒеҖјпјҢиҝҷдёӘе°ұи·ҹе…·дҪ“зҡ„и®ҫи®Ўжңүе…ідәҶпјҢиҝҷйҮҢдёҚиҝҮеӨҡиөҳиҝ°гҖӮ
